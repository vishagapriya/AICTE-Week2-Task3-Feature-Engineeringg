# Week 2 Task 3 - Feature Engineering and Selection

For this task, I worked on the Titanic dataset and performed feature engineering and feature selection using Python.

First, I handled the missing values present in the dataset and converted categorical columns into numerical values so that they could be used for machine learning. I then created new features such as FamilySize and IsAlone from the existing columns.

After preprocessing the data, I applied feature scaling and used SelectKBest to identify the most important features related to passenger survival.

This task helped me understand how feature engineering can improve the quality of data and how feature selection helps in choosing the most relevant features for building better machine learning models.
